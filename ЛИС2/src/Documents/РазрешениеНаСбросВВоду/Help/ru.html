Документ «Разрешение на сброс ЗВ в воду»

В документе «Разрешение на сброс ЗВ в воду» содержатся сведения о выданных организации разрешениях на выброс загрязняющих веществ в воду.
Форму списка документа можно открыть, выбрав на панели разделов пункт «Разрешение на сброс ЗВ в воду» в разделе «Охрана водных объектов».
Будет открыта форма списка документа «Разрешение на сброс ЗВ в воду».
Для открытия формы документа необходимо дважды щелкнуть на соответствующей строке в форме списка.
Для добавления нового документа необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента документа «Разрешение на сброс ЗВ в воду». Поля, обязательные для заполнения отмечены красным подчеркиванием. Основные реквизиты:
Регистрационный номер – указывается регистрационный номер разрешения;

  • Дата выдачи – дата выдачи разрешения;
  • Надзорный орган, выдавший документ;
  • Описание разрешительного документа;
  • Периодичность разрешения – по годам или по кварталам;
  • Организация – организация, которой получено разрешение;
  • Производственная площадка – площадка, по которой выдано разрешение;
  • Документ-основания – указывается документ право водоотведения;
  • Признак Отображение ВСС – устанавливается флаг, если в рамках разрешения определяется временно-согласованный сброс.

На закладке «Нормативы по выпускам» содержится две табличных части: «Разрешение по выпускам» – содержит сведения о выпусках и разрешенных для них сбросам; «Перечень загрязняющих веществ» – содержит сведения о загрязняющих веществах, на сброс которых выдано разрешение по текущему выбранному выпуску в таблице разрешений по выпускам.

Данные с указанной вкладки попадают в план-график ПЭК на вкладку «Контроль сточных вод». По этим данным заполняются нормативы для точек контроля, которые расположены на выпусках. Также на вкладке указывается норматив допустимого сброса на период действия разрешения.
После завершения формирования документа «Разрешение на сброс ЗВ в воду», его необходимо провести. Проведение документа производится кнопкой "Провести", расположенной на командной панели формы документа. Форма документа остается открытой. Также проведение возможно кнопкой "Провести и закрыть", расположенной на командной панели формы документа. Форма документа закрывается.
