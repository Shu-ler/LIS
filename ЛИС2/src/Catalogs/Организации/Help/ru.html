Организации

В справочнике «Организации» хранятся сведения, характеризующие предприятия, ведущие природоохранную деятельность.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «Организации» в разделе «Предприятие».
Будет открыта форма списка справочника «Организации».
Для открытия формы организации необходимо дважды щелкнуть на соответствующей строке в форме списка. Для добавления новой организации необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Организации».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Основные реквизиты:
Наименование – краткое название организации, данная строка будет являться основным текстовым представлением организации в Системе, в том числе данное наименование будет отображаться в форме списка справочника «Организации».
Юр./Физ. лицо – выбор организационно-правовой формы предприятия. Состав реквизитов, описывающих Юридическое лицо, отличается от состава реквизитов, описывающих Физическое лицо;
Полное наименование – полное название предприятия как оно указано в учредительных документах;
Расчетный счет – номер банковского расчетного счета предприятия;
Банк – банковское учреждение, в котором обслуживается предприятие, выбирается из справочника «Банки»;
Руководитель, Главный бухгалтер – ответственные лица, чьи подписи будут стоять на различных документах и отчетах, формируемых в программе. Выбираются из справочника «Сотрудники».
Юридический адрес, Фактический адрес – адреса предприятия. Могут вводиться пользователем напрямую в поле ввода, либо по нажатию кнопки выбора, ввести составляющие адреса по отдельности в появившемся окне. По нажатию кнопки "ОК", составляющие сформируются в полную строку адреса, при этом, если в адресной строке уже были введены данные, они будут заменены. По нажатию кнопки Отмена адресная строка останется без изменений.

Для завершения ввода данных организации необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
