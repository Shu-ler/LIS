Банки

 

Справочник «Банки» предназначен для хранения реквизитов банковских учреждений, в которых обслуживаются предприятие пользователя и предприятия контрагентов. Окно списка контрагентов вызывается из главного меню командой Справочники -> Банки. Для добавления нового банка используется кнопка . При выводе диалогового окна необходимо заполнить поля, необходимые для заполнения банковских реквизитов платежных документов. Ввиду возможных изменений банковских реквизитов справочник «Банки» периодически необходимо обновлять.

Основные реквизиты:

  • Наименование банка - название банка так, как оно указывается в платежных документах;
  • Местонахождение - город, в котором находится банковское отделение;
  • БИК - банковский идентификационный код, состоящий из девяти цифр;
  • Корр. счет - корреспондентский счет банка.
