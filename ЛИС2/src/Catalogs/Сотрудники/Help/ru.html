Сотрудники

В справочнике хранится информация о сотрудниках организации.
Справочник может включать два уровня иерархии. Это позволяет группировать сотрудников по категориям и быстро их находить.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «Сотрудники» в разделе «Предприятие».
Будет открыта форма списка справочника «Сотрудники».
Для создания новой группы можно нажать на кнопку "Создать группу" на командной панели формы списка «Сотрудники». При вводе новой группы программа предлагает указать наименование группы. Также можно указать, в какую группу должна включаться вновь вводимая группа.
Для открытия формы сотрудника необходимо дважды щелкнуть на соответствующей строке в форме списка.
Для добавления нового сотрудника необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Сотрудники».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Основные реквизиты:
Ф.И.О. – полные фамилия, имя, отчество сотрудника;
Должность – должность сотрудника на предприятии;
Действует по доверенности – признак, который устанавливается в случае, если организация является обособленным подразделением, устанавливается для должности бухгалтера или руководителя. Если галочка установлена, необходимо заполнить поле «Реквизиты доверенности».
В случае, если установлена галочка «Ответственный за экологию» на форме отображается дополнительная вкладка «Обучение по пром. экологии». На данной вкладке необходимо заполнить данные по соответствующему обучению сотрудника.
В случае, если установлена галочка «Ответственный за обращение с отходами» на форме отображается дополнительная вкладка «Обучение по обращению с отходами». На данной вкладке необходимо заполнить данные по соответствующему обучению сотрудника.
Для открытия списка присоединенных файлов необходимо использовать гиперссылку "Присоединенные файлы" в шапке формы.
Для завершения ввода данных сотрудника необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
