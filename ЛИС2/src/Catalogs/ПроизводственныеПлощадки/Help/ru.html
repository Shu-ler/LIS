Производственные площадки

Справочник содержит список производственных площадок предприятия.
Справочник может включать три уровня иерархии. Это позволяет группировать площадки по категориям и быстро их находить.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «Контрагенты» в разделе «Производственные площадки».
Будет открыта форма списка справочника «Производственные площадки». На форме списка отображается иерархия организаций и их производственных площадок. В левой части формы отображаются организации, в правой части формы, с отбором по выбранной организации, отображаются площадки, а также связанные с ними участки и цеха.
Для открытия формы производственной площадки необходимо дважды щелкнуть на соответствующей строке в правой части формы списка.
Для добавления новой производственной площадки необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Производственные площадки».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Необходимо ввести наименование производственной площадки, а также ее вид.
Форма содержит несколько вкладок, содержащих сведения о данной производственной площадке.
Общее – общие сведения о производственной площадке.
Хранимые отходы – сведения об отходах, хранимых на площадке в настоящий момент времени, а также историю движений отходов по площадке. Для отображения списка хранимых отходов с учетом отходов, содержащихся на подчиненных участках необходимо нажать кнопку "Показывать с учетом подчиненных участков".
Характеристики – сведения о характеристиках производственной площадки.
Выпуски в воду – сведения о выпусках в воду данной производственной площадки.
Для завершения ввода данных производственной площадки необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
