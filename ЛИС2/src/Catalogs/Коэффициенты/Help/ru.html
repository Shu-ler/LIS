Коэффициенты и вспомогательные величины

 

Справочник «Значение констант» служит для хранения информации, которая либо совсем не изменяется в процессе функционирования программы, либо меняется достаточно редко. Наиболее простой пример – плотность отработанного масла, которое, как правило, не меняется. Вызывается через главное меню Справочники -> Значения констант.

Константы предназначены для того, чтобы один раз занести информацию, а затем использовать ее при формировании документов, в расчетах, при построении отчетных форм. При каких-либо изменениях плотности масла достаточно изменить его всего один раз – в константе – и все изменения автоматически будут отражены в тех местах, где эта константа используется.

Все константы, необходимые для работы, уже внесены в справочник. При изменении значения константы в нижнюю таблицу "История изменений значения констант" будет добавлена строчка с новым значением, а начало действия нового значения автоматически устанавливается начиная с начала текущего квартала. Для изменения даты, начиная с которой будет действовать значение константы, необходимо изменить поле Период в нижней таблице.

Внимание! При начале работы с конфигурацией "Охрана окружающей среды" необходимо сразу заполнить все значения констант, начиная с даты, от которой вы будете в дальнейшем формировать отчеты. Если константы не будут заполнены, при формировании некоторых отчетов будет выдаваться ошибка. В расчетах всегда участвуют константы, даты которых более поздние (однако не позже конца периода, за который формируется отчет).

Примечание: значение Коэффициента, учитывающего инфляцию может быть разным для различных загрязняющих веществ и отходов. Укажите в данном справочнике самое распространенное значение. Ввод отличных от введенного значений для веществ и отходов осуществляется через окно Справочники -> Индексация коэффициентов.
