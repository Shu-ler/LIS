Договоры контрагентов

Для открытия формы договора необходимо дважды щелкнуть на соответствующей строке в списке, расположенном на вкладке «Договоры» формы элемента справочника "Контрагенты".
Для добавления нового договора необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Договоры контрагентов».
На форме необходимо заполнить данные договора: номер, дату, вид заключенного договора, организацию, статус, срок действия, указание заключен ли он с пролонгацией или без нее, внести крайний срок оплаты по договору. Также нужно указать номер и дату лицензии контрагента.
Для открытия списка присоединенных файлов необходимо использовать гиперссылку "Присоединенные файлы" в шапке формы.
Для завершения ввода данных договора необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
Для установки договора основным для данного контрагента необходимо нажать кнопку "Признак основного договора"  на вкладке «Договоры» формы контрагента. Основным будет установлен текущий выделенный в списке договор, значения полей договора в списке будут выделены жирным шрифтом.
При установке галочки «Осуществляет плату за НВ на ОС» будет отображена дополнительная таблица, в которой следует заполнить данные о сроках оплаты контрагента за негативное воздействие на окружающую среду.
Для завершения ввода данных контрагента необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
