Водозаборы

В справочнике «Водозаборы» содержатся сведения об источниках водозабора предприятия.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «Водозаборы» в разделе «Вода».
Будет открыта форма списка справочника «Водозаборы».
Для создания новой группы можно нажать на кнопку "Создать группу" на командной панели формы списка «Водозаборы». При вводе новой группы программа предлагает указать наименование группы. Также можно указать, в какую группу должна включаться вновь вводимая группа.
Для открытия формы водозабора необходимо дважды щелкнуть на соответствующей строке в форме списка.
Для добавления нового водозабора необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Водозаборы».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Основные реквизиты:
Наименование – наименование водозабора.
Площадка – производственная площадка, к которой относится водозабор.
Фактический адрес – фактический адрес водозабора.
Водный объект – водный объект, к которому относится водозабор.
Дата введения в эксплуатацию – дата ввода водозабора в эксплуатацию.
Для завершения ввода данных водозабора необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
