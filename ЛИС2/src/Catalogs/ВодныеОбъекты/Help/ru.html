Водные объекты

В справочнике «Водные объекты» содержатся сведения о водных объектах.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «Водные объекты» в разделе «Вода».
Будет открыта форма списка справочника «Водные объекты».
Для создания новой группы можно нажать на кнопку "Создать группу" на командной панели формы списка «Водные объекты». При вводе новой группы программа предлагает указать наименование группы. Также можно указать, в какую группу должна включаться вновь вводимая группа.
Для открытия формы водного объекта необходимо дважды щелкнуть на соответствующей строке в форме списка.
Для добавления нового водного объекта необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Водные объекты».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Основные реквизиты:
Наименование – наименование водного объекта.
Водохозяйственный участок – водохозяйственный участок, к которому относится водный объект.
Таблица Проверки – содержит данные о проведенных проверках.
Для завершения ввода данных водного объекта необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
