Ввод информации об источнике загрязнения

Реквизиты справочника изменяются в зависимости от выбранного типа источника загрязнения. Существует два типа источника загрязнения:

  • Организованный
  • Неорганизованный

После того как все реквизиты элемента справочника будут заполнены, элемент необходимо записать по кнопке «Записать».

При наличии зарегистрированного в системе разрешения на выброс загрязняющих веществ по данному источнику загрязнения на форме справочника появляется дополнительная закладка «Установленные нормативы выбросов по ИЗА», содержащая перечень загрязняющих веществ, на выброс которых получено разрешение, с указанием значения предельно-допустимого выброса для каждого из них.
