Вещества, загрязняющие воду

В справочнике «Вещества, загрязняющие воду» содержатся сведения о веществах, загрязняющих водные объекты.
Справочник может включать два уровня иерархии. Это позволяет группировать загрязняющие вещества по категориям и быстро их находить.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «Вещества, загрязняющие воду» в разделе «Вода».
Будет открыта форма списка справочника «Вещества, загрязняющие воду».
Для создания новой группы можно нажать на кнопку "Создать группу" на командной панели формы списка «Вещества, загрязняющие воду». При вводе новой группы программа предлагает указать наименование группы. Также можно указать, в какую группу должна включаться вновь вводимая группа.
Для открытия формы загрязняющего вещества необходимо дважды щелкнуть на соответствующей строке в форме списка. Для добавления нового загрязняющего вещества необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Вещества, загрязняющие воду».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Основные реквизиты:
Единица измерения – единица, которой измеряется загрязняющее вещество.
Сумма платы ПДС, руб./тонну, Сумма платы ПДС, руб./тонну – сумма денежных выплат за тонну текущего вещества.
Для завершения ввода данных загрязняющего вещества необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
