Контрагенты

В процессе финансово-хозяйственной деятельности учреждение вступает в отношения с различными контрагентами — организациями. Для хранения информации о них предназначен справочник «Контрагенты».
Справочник может включать три уровня иерархии. Это позволяет группировать контрагентов по категориям и быстро их находить.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «Контрагенты» в разделе «Предприятие».
Будет открыта форма списка справочника «Контрагенты».
Для создания новой группы можно нажать на кнопку "Создать группу" на командной панели формы списка «Контрагенты». При вводе новой группы программа предлагает указать наименование группы. Также можно указать, в какую группу должна включаться вновь вводимая группа.
Для открытия формы контрагента необходимо дважды щелкнуть на соответствующей строке в форме списка.
Для добавления нового контрагента необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Контрагенты».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Основные реквизиты:
Юр.лицо/ИП – организационно правовая форма предприятия-контрагента.
Полное наименование – полное наименование предприятия-контрагента. В это виде название предприятия будет указано в формируемых вами печатных формах и отчетах;
Расчетный счет – номер банковского расчетного счета контрагента;
Банк – банковское учреждение, в котором обслуживается контрагент, выбирается из справочника «Банки»;
Контактное лицо – контактная информация представителя контрагента; При нажатии на гиперссылку "Контактное лицо" будет отображен набор полей, которые необходимо заполнить информацией о контактных данных представителя контрагента.
Юридический адрес, Фактический адрес – адреса контрагента. Могут вводиться пользователем напрямую в поле ввода, либо по нажатию кнопки выбора, ввести составляющие адреса по отдельности в появившемся окне. По нажатию кнопки "ОК", составляющие сформируются в полную строку адреса, при этом, если в адресной строке уже были введены данные, они будут заменены. По нажатию кнопки "Отмена", адресная строка останется без изменений.
Для открытия списка присоединенных файлов необходимо использовать гиперссылку "Присоединенные файлы" в шапке формы.
В случае, если установлена галочка «Является полигоном отходов с собственным ОКТМО» на форме отображается дополнительная вкладка «Полигон». На данной вкладке необходимо заполнить данные полигона.
В случае, если установлена галочка «Учебный центр/лаборатория» на форме отображается дополнительная вкладка «Уч.центр/лаборатория». На данной вкладке необходимо заполнить данные лаборатории.
Информация по договорам контрагента содержится на вкладке «Договоры».
Для открытия формы договора необходимо дважды щелкнуть на соответствующей строке в списке.
Для добавления нового договора необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «Договоры контрагентов».
На форме необходимо заполнить данные договора: номер, дату, вид заключенного договора, организацию, статус, срок действия, указание заключен ли он с пролонгацией или без нее, внести крайний срок оплаты по договору. Также нужно указать номер и дату лицензии контрагента.
Для открытия списка присоединенных файлов необходимо использовать гиперссылку "Присоединенные файлы" в шапке формы.
Для завершения ввода данных договора необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
Для установки договора основным для данного контрагента необходимо нажать кнопку "Признак основного договора"  на вкладке «Договоры» формы контрагента. Основным будет установлен текущий выделенный в списке договор, значения полей договора в списке будут выделены жирным шрифтом.
При установке галочки «Осуществляет плату за НВ на ОС» будет отображена дополнительная таблица, в которой следует заполнить данные о сроках оплаты контрагента за негативное воздействие на окружающую среду.
Для завершения ввода данных контрагента необходимо записать элемент справочника.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
