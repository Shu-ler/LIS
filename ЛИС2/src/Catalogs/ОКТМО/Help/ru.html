ОКТМО

Справочник содержит общероссийский классификатор территорий муниципальных образований.
Справочник может включать четыре уровня иерархии.
Форму списка справочника можно открыть, выбрав на панели разделов пункт «ОКТМО» в разделе «Предприятие».
Будет открыта форма списка справочника «ОКТМО».
Для открытия формы территории муниципального образования необходимо дважды щелкнуть на соответствующей строке в правой части формы списка.
Для добавления новой территории муниципального образования необходимо нажать кнопку "Создать" на командной панели формы списка. Будет открыта форма элемента справочника «ОКТМО».
Поля, обязательные для заполнения отмечены красным подчеркиванием.
Необходимо ввести наименование территории муниципального образования, а также ее код.
Основные реквизиты:
Наименование – наименование территории муниципального образования.
Код по ОКТМО – код территории муниципального образования.
Округ – округ присвоения ОКТМО.
Дополнительный коэффициент 1,2 – дополнительный коэффициент 1,2 расчета платы за НВ на ОС по ОКТМО.
Дополнительный коэффициент 2– дополнительный коэффициент 2 расчета платы за НВ на ОС по ОКТМО.
Коэффициент экологической значимости– коэффициент экологической значимости расчета платы за НВ на ОС по ОКТМО.
Коэффициент экологической значимости для почвы– коэффициент экологической значимости для почвы расчета платы за НВ на ОС по ОКТМО.
Код субъекта РФ– код субъекта РФ, по которому присвоен ОКТМО.
Запись производится кнопкой "Записать", расположенной на командной панели формы элемента справочника. Форма остается открытой.
Также запись возможна кнопкой "Записать и закрыть", расположенной на командной панели формы. Форма элемента справочника закрывается.
